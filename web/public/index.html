Composable sandboxes for agent software

CPSL

Package tools, permissions, files, network access, and language adapters into explicit sandbox images. The same runtime contract can live in a CLI, server, app host, or browser.

Install

cpsl-web

Capsule built with cpsl-web.toml Allowed domains httpbin.org

Small Capsules

Build only the modules a task needs and leave everything else outside the runtime.

Policy First

Filesystem mounts, HTTP domains, and native capabilities are selected deliberately.

Agent Native

Run Bash-style commands, a Python-compatible subset, or direct Luau in one sandbox.

From source

Install CPSL

git clone https://github.com/fundamental-research-labs/cpsl
cd cpsl
./build-cli.sh
./cpsl -i
